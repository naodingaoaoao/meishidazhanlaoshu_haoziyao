# 美食大战老鼠改卡辅助

## 简介
美食大战老鼠改卡辅助是一款基于易语言开发的辅助工具，旨在帮助玩家在《美食大战老鼠》游戏中进行改卡操作。严禁出售本项目，请您在合法合规的前提下使用。
本人只是个技术菜鸡，希望各位大佬指正提携

## 功能
- 改卡辅助功能
- 平台切换
- 一键静音
- 缓存清除
- cookie清除

## 安装
1. 下载项目源码至本地。放在开源项目sunny与FBrowserCEF3lib项目同目录下！
2. 使用易语言编译器打开项目，编译并生成可执行文件。
